# Cascade Travel Landing Page

This is a modern, single-page website for a travel brand called "Cascade." I built this to practice clean UI/UX design and centered hero layouts.

### What’s in it:
* **Sticky Navbar:** A semi-transparent menu at the top that stays as you scroll.
* **Hero Section:** Large, centered text with a "Travel Places" subtitle.
* **Call-to-Action Buttons:** Red "Explore" and "Subscribe" buttons with smooth hover effects.
* **Responsive Background:** A full-height (100vh) background that covers the whole screen.

### Tech Stack:
* **HTML5** - Page structure.
* **CSS3** - Positioning, transitions, and Google Fonts integration.

### Project Files:
* `index.html` - Main page code.
* `style.css` - All the design and layout rules.
* `bg1.jpg` - The background image.
* `logo.png` - The brand logo.
